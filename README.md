 Overview

WomenConnect is a full-stack web application designed to manage women-led organizations efficiently. It provides role-based dashboards for admins and members, enabling communication, financial tracking, and access to a marketplace.


 Tech Stack

Frontend

React.js
HTML, CSS, JavaScript
Axios

Backend

Node.js
Express.js

Database
MongoDB

Tools
Git & GitHub

⚙️ Installation & Setup
1. Clone the Repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
2. Install Dependencies

Backend

cd server
npm install

Frontend

cd client
npm install
3. Setup Environment Variables

Create a .env file inside the server folder:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
4. Run the Project

Start Backend

cd server
npm start

Start Frontend

cd client
npm start
